== Description ==

= Descrição =

Altere suas URLs internas (Slug) dos seus Post Type personalizado em relação ao idioma desejado do sistema.

Aproveite a oportunidade de mudar o URL interna (Slug) seus Post Type personalizado, vinculando ao idioma desejado no sistema Polylang ou WordPress. Muito fácil!

= Exemplo =
* Se o idioma for Português Brasil a URL interna do Post Type Custom Equipe será (pt/equipe/carlosramosweb/).
* Se o idioma for Inglês USA a URL interna do Post Type Custom Equipe será (en/team/carlosramosweb/).
* A mudança de equipe para team, ou vice e versa, se torna necessário e ajuda no SEO.
* Isso somente alterando na configuração do WordPress, sem você precisar entender de codificação.
* Ao desativar o plugin, tudo voltará ao normal.
* Ao ativar, as configurações anteriores que você deixou retornará.
* Se você excluir o plugin do diretório, aí sim, apagará as configurações e tudo retorna quando antes do plugin ser ativado.


== Installation ==

= Instalação =
* Faça o upload para o diretório /wp-content/plugins/.
* Ative o plugin através do menu ‘Plugins’ no WordPress.
* Vá para o painel de configuração Ferramentas > Slug Language.
